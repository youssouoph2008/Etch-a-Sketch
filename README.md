# Etch-a-Sketch / Pixel Sketchpad
# Description

This project is a small browser-based Sketchpad that lets you draw on a grid of cells using JavaScript and the DOM.

Users can:

Create an initial 16x16 grid.
Change the number of cells per side using a button.
Draw in black when hovering over cells.
Draw with random colors using a "Random Color" button.

This project is designed to practice DOM manipulation, event listeners, and CSS Flexbox.

# Technologies Used
HTML5
CSS3 (Flexbox)
JavaScript (DOM, event listeners, prompts)
How to Use
Open index.html in your browser.
You will see a default 16x16 grid.
Available buttons:
New Grid: Creates a new black grid.
Random Color: Creates a new grid where cells change to random colors when hovered.
You will be prompted to enter the number of cells per side (maximum 100).
Hover over the cells to draw.

# Features
Dynamic creation of cells with JavaScript.
Automatic adjustment of cell size based on the number of cells per side.
Hover effect to draw on the grid.
Random color generation for fun effects.
Maximum of 100 cells per side to prevent performance issues.
